# Sanity Studio - {{PROJECT_NAME}}

Content management system powered by Sanity.io

## Development

```bash
npm run dev
```

Open [http://localhost:3333](http://localhost:3333)

## Deploy Studio

```bash
npm run deploy
```

Your studio will be available at `https://your-project.sanity.studio`

## Schemas

Add schemas in `schemas/` directory and register them in `schemas/index.ts`

## Configuration

See `sanity.config.ts` for studio configuration.
